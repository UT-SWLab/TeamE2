About


What is this site?

This site displays Formula 1 statistics, aggregated from multiple public APIs. The site features hisotrical and current data for things ranging from drivers to constructors to circuits. The goal of the website is to combine all these components into one easy to access site. Users that would benefit from this site include Formula 1 team members, Formula 1 fans, sport analysts, or anyone else looking to improve their understanding of Formula 1.

What did we find out?

By integrating the data from multiple sources into one cohesive locataion, we hope to produce an accurate analysis of Formula 1 trends, including up-and-coming drivers, most successful constructors, and circuit timings. With enough data, the team might even be able to make predictions about future races. Data Sources

The data was taken from public Formula 1 REST APIs using the Python requests library. The HTTP requests retrieve the relevant data in JSON form, which is then parsed into our own database structure. The data is then displayed to the user here on the site. More information will be added here, especially concerning how the data is stored, in the next phase(s).
